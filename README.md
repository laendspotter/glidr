# glidr

social-app für segelflug-piloten. teile deine flüge, sieh wer gerade in der luft ist, verfolge distanz- und höhen-stats, verbinde dich mit anderen piloten.

live-tracking kombiniert OGN-daten (start-/landungserkennung) mit GPS (genauigkeit während des flugs). vereinsliste und OLC/WeGlide-wertungen werden direkt über die öffentliche WeGlide-API geladen.

gebaut mit React Native (Expo) und Firebase.
